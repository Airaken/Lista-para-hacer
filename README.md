## test de Interfell

1. Montar la base de datos en pgAdmin

restura la base de datos con el archivo
```
restore_test.sql
```

2. Instalar paquetes

```
npm install
```

3. Correr el comando start

```
npm start
```
#NOTA:
en caso de que no este generado el archivo bundle.js en la ruta public/dist, correr antes el comando
```
npm run webpack
```
